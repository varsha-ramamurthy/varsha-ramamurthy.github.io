    • Summary: Latitude vs Longitude

    This website shows an analysis of global climate. Using the data acquired by performing API calls using OpenWeatherMap, we were able to get data on climate conditions in diffewrent cities across the world. Comparisons were made between latitude and other climate-influenced factors such as temperature, humidity, cloudiness and wind-speed. It was found that more tropical conditions are observed closer to the equator, and extreme conditions observed towards the poles.
